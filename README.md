## Installation

Clone the repository:

```bash
git clone https://github.com/pruthi-shivin/Webnest_RotarAI
```

Navigate into the project folder:

```bash
cd Webnest_RotarAI
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
